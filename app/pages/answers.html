Что такое DOM?

DOM или Document Object Model - это объектная модель с помощью которой документы XML/HTML преобразовываються в дерево тегов, доступное для преобразования через JS.

Что такое узел (нода)? Какие существуют ноды?

Теги и тексты внутри них образовывают узлы-элементы(element node) и дочерние текстовые узлы(text node), соотвественно.

Какие есть способы поиска элементов на странице?

Существует поиск элемента по его классу, id и селектору.

Что возвращает метод document.getElementById()? А метод document.querySelectorAll()?

Первый метод вернёт элемент с соответвующим id, второй выведет коллекцию элементов с определенным селектором.

Как изменить стили элемента? Какая существует особенность с правилам, в названии которых есть дефис (border-radius, padding-left)?

Для изминения стилей используют свойство element.style. CSS-свойство следует записывать через CamelCase.

Чем отличается коллекция от массива?

Коллекции так же называют "псевдомассивами". Разница в том, что такие методы массивов как forEach, map, push, pop и другие, не работают с коллекциями.

Как обратиться к родителю элемента? А к потомку?

Что бы обратиться к родителю используют синтаксис element.parentNode, к потомку element.childNodes.

Опиши, какие методы используются для работы с классами? Что они делают?

elem.className свойство через которое можно вернуть значение атрибута class элемента или задать своё. elem.classList это свойство схоже с преддыдущим, но даёт возможность работать с целой коллекцией классов элемента. Так же существует метод getElementsByClassName() с помощью которого можно найти элемент или коллекцию элементов принадлежащих заданому классу.

Что находится в свойстве Element.innerHTML?

В данном свойстве содержиться значение дочерних элементов указаного узла.

Как работает метод Element.matches()?

Метод определяет принадлежит ли элемент указаному селектору и возвращает значение true или false соотвественно.

Зачем нужна библиотека Modernizr?

Данная бибилиотека помогает определить поддерживает ли указанный браузер те или иные свойства HTML5 и CSS3.

Какие методы создают новый элемент, текстовую ноду?

document.createElement(tag) для нового элемента и document.createTextNode(text) для создания текстовой ноды.

Чем отличаются Node.appendChild(), Node.insertBefore() и Element.insertAdjacentHTML()?

Отличие заключаеться в том что первый метод добавит элемент в конец списка родителя, второй метод добавит элемент в начало списка родителя, а в третьем методе можно указать положение элемента (первый, после первого, предпоследний, последний).

Какие два метода используются для удаления элементов?

element.removeChild(child) удалит дочерний элемент из узла. Node.remove() удалит ноду из дерева DOM.
