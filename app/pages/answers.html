Что такое EcmaScript?

Это язык программирования, который является основой для построения других Script языков.

Зачем нужны функции?

Функция задаёт действие, потом с помощью её вызова мы можем выполнять данное действие несколько раз и в любой части программы.

Сколько параметров можно передать функции?

Функцию можно вызвать с любым количеством аргументов.

Что такое область видимости?

Область видимости определяет область в которой будет виден объект. К примеру переменные заданные в теле функции будут действовать только приделах этой функции, для других мест программы они останутся невидимыми.

Чем отличается Function Declaration от Function Expression?

Function Declaration - объявляется в основном потоке кода. Их можно вызывать до непосредственного объявления, так как создются интерпретатором до выполнения кода.
Пример:
function sum(a, b) {
return a + b;
}
Function Expression - объявляется через какое либо выражение.
Пример:
var sum = function(a, b) {
return a + b;
}

Что такое замыкание?

Это инструмент который использует возможности зоны видимости. С помощью этого приёма в функции можно вызывать переменную из области в которой находиться она сама.
Пример:
function numberGenerator() {
var num = 1;
function checkNumber() {
console.log(num);
}
}

Как можно добавить свойства или методы у объекта? Как можно удалить свойство или метод?

Для добавления свойства или метода используют запись вида объект.свойство/метод. Для удаления свойства или объекта используют оператор delete: delete объект.свойство/метод.

Как создать свойство только для чтения?

Для создания такого свойства следует использовать метод Object.defineProperty. С помощью дескрипторов данного метода мы можем задать поведение свойству. В нашем случае мы используем дескриптор writable. Он определяет возможности изменения значения свойства. Что бы свойство было только для чтения, в дескрипторе указываем false.

Что такое ООП? Какие ты знаешь принципы ООП?

ООП или объектно ориенторованое програмирование, при котором функции и данные группируються в "объекте". При ООП в каждом объекте описываються взаимодействие объектов между собой и со страницей браузера, а так же смена их свойств.

Что такое "функциональное программирование"? Чем отличается ООП и ФП? Какие есть функциональные библиотеки в JavaScript?

ФП это программирование в основе которого лежит формальная система Лямбда-исчисления. Отличие состоит в том, что в ФП вычисление производят с помощью результатов функций от исходных данных и результатов других функций. Underscore.js

Что такое прототипное наследование?

Каждый объект содержит в себе ссылку на другой объект, который в свою очередь называют прототипом. Такие прототипы могут иметь свой прототип. Данная цепочка завершиться в случае, если свойство последующего прототипа будет равно null. С помощью наследования прототип может получать доступ к методам основного объекта.

Как можно обратиться к свойству с именем my-SoMe-PROPERTY+name?

Данное имя вызовет ошибку из-за использования знаков "-" и "+". Для воизбежания такой ситуации следует использовать CamelCase. Верное имя будет выглядить вот так mySoMePROPERTYname. А образение будет иметь вид Object.mySoMePROPERTYname.

Объясни, что такое деструктуризация.

Деструктуризация - синтаксис при котором массив или объект разбивают на части и таким образом присваивают несколько переменных.

Что такое DOM?

DOM или Document Object Model - это объектная модель с помощью которой документы XML/HTML преобразовываються в дерево тегов, доступное для преобразования через JS.

Что такое узел (нода)? Какие существуют ноды?

Теги и тексты внутри них образовывают узлы-элементы(element node) и дочерние текстовые узлы(text node), соотвественно.

Какие есть способы поиска элементов на странице?

Существует поиск элемента по его классу, id и селектору.

Что возвращает метод document.getElementById()? А метод document.querySelectorAll()?

Первый метод вернёт элемент с соответвующим id, второй выведет коллекцию элементов с определенным селектором.

Как изменить стили элемента? Какая существует особенность с правилам, в названии которых есть дефис (border-radius, padding-left)?

Для изминения стилей используют свойство element.style. CSS-свойство следует записывать через CamelCase.

Чем отличается коллекция от массива?

Коллекции так же называют "псевдомассивами". Разница в том, что такие методы массивов как forEach, map, push, pop и другие, не работают с коллекциями.

Как обратиться к родителю элемента? А к потомку?

Что бы обратиться к родителю используют синтаксис element.parentNode, к потомку element.childNodes.

Опиши, какие методы используются для работы с классами? Что они делают?

elem.className свойство через которое можно вернуть значение атрибута class элемента или задать своё. elem.classList это свойство схоже с преддыдущим, но даёт возможность работать с целой коллекцией классов элемента. Так же существует метод getElementsByClassName() с помощью которого можно найти элемент или коллекцию элементов принадлежащих заданому классу.

Что находится в свойстве Element.innerHTML?

В данном свойстве содержиться значение дочерних элементов указаного узла.

Как работает метод Element.matches()?

Метод определяет принадлежит ли элемент указаному селектору и возвращает значение true или false соотвественно.

Зачем нужна библиотека Modernizr?

Данная бибилиотека помогает определить поддерживает ли указанный браузер те или иные свойства HTML5 и CSS3.

Какие методы создают новый элемент, текстовую ноду?

document.createElement(tag) для нового элемента и document.createTextNode(text) для создания текстовой ноды.

Чем отличаются Node.appendChild(), Node.insertBefore() и Element.insertAdjacentHTML()?

Отличие заключаеться в том что первый метод добавит элемент в конец списка родителя, второй метод добавит элемент в начало списка родителя, а в третьем методе можно указать положение элемента (первый, после первого, предпоследний, последний).

Какие два метода используются для удаления элементов?

element.removeChild(child) удалит дочерний элемент из узла. Node.remove() удалит ноду из дерева DOM.
