Что такое EcmaScript?

Это язык программирования, который является основой для построения других Script языков.

Для чего используются и чем отличаются атрибуты async и defer?

Эти два специальных атрибута. Они асинхронно загружают срипт.
Аsync не ожидает полной загрузки HTML-страницы, это влияет на последовательность выполнения скриптов.
Defer всегда запускает скрипты в четком порядке после полной загрузки страницы.

Почему предпочтительнее подключать скрипты перед /body, а не в head?

Загрузка и отображение HTML всегда выполняется построчно. При подключении скрипта в head, сначала будет выполнятся загрузка скрипта, а потом уже остальной части страницы. Это может создать некий колапс, если наши скрипты внешние. Ожидание загрузки страницы увеличится, что негативно повлияет на оценку пользователя. Для во избежания такой ситуации, скрипт подключают перед закрывающим тегов /body

Какие основные отличия интерпретируемого языка программирования от компилируемого?

Компилированные языки приближены к машному коду, по этому нуждаются в дополнительном компилировании перед интерпретацией.
Интерпретируемые языки обрабатываются и выполняются интерпретатором сразу.

Что такое языки высокого и низкого уровня? Какие ты знаешь?

Язык программирования высокого уровня содержит смысловые конструкции и команды, которые представляют из себя стандартные структуры из нескольких простейших (низкоуровневых, машинных) команд, таким образом программист освобождается от необходимости писать каждую машинную команду по отдельности, то есть сокращается время работы программиста для написания определенного функционала, также сокращается размер текстового файла содержащего исходный код алгоритма программы, команды выглядят более человеко понятными и могут объединяться в структуры (блоки кода из нескольких команд), всё это способствует возможности "держать в голове" весь алгоритм программы, работать с небольшими и понятными областями кода взамен огромных листов утомительных машинных кодов, которые включает язык программирования низкого уровня.
Пример языков высокого уровня: Python, Ruby, JavaScript, Common Lisp.
Пример языков низкого уровня:C, C++, Assembler.

Для чего используются переменные? Какие ключевые слова используются для создания переменных? Какие есть отличия в переопределении значений при задании переменной через let, const и var?

Переменные помогают программе аппелировать полученными данными, сохраняя их в отдельных областях памяти. Существуют такие слова для назначения переменных как var, let, const.
Var применяют для назначения переменной глоабльно.
Let и const действует в пределах одного блока. Константа обязательно определенны со значением и не могут изменяться впоследствии.

Нет, имя переменной не может начинаться с цифры или не разрещенных символов.

Какие слова запрещено использовать в названии переменной?

Запрещенно использовать командные слова такие как var, export, class, return и др.

Что такое СamelCase и snake_case?

Это виды написания слов. В первом случае, все слова пишутся слитно и каждое новое с большой буквы. Во втором, слова пишут только маленькими буквамы, а пробелы заменяют нижним подчеркиванием "_".

Что такое "типы данных"? Какой оператор позволяет определить тип данных?

Типы данных в языке программирования JavaScript, числа, строки, логические значения true и false, отличие null от undefined. Для определения типа данных используют оператор typeof.

Что такое "динамическое приведение типов данных"? Можно ли писать JavaScript со статическим приведением типов?

Динамическое приведение типов данных - преобразования оператора из одного типа в другой интерпретатором или самим програмистом. В JavaScript можно добиться статичности типов только с помощью сторонних инструментов как Flow b TypeScript.

Что такое "алгоритм"? Какие ты знаешь типы алгоритмов?

Алгоритм это последовательность действий, которое всегда имеет цель. Линейные, цикличные, с ветвлениями.

Зачем нужны циклы? Какие виды циклов ты знаешь? В чем их основные отличия?

Циклы описывают действие, его повторение будет продолжаться указаное число раз пока не будет выполнено данное задание. Существуют циклы с предусловием и циклы с пост условием. Циклы с предусловие сначала выполняют проверку заданного условия, при его выполнении запускаются повторяющие действия. В цикле с пост условием программа сначала единожды выполнит те действия, которые должны повторяться, после проверит логическое условие выхода из цикла.

Что деляют операторы ++i и i++ и в чем их отличие?

Данные операторы увелицивают переменную на единицу. Разница состоит в том, что первый оператор сначала выполнит действие с переменной, а потом выводит результат этого дествия. Второй выведет переменную, а потом совершит действие с ней.

Что такое "итерация"?

Итерацией называют один шаг цикла, то есть одно его повторние.

Как называются и чем отличаются операторы || и &&?

Данные операторы называют логическими. Оператор || отображает знаечние ИЛИ. Если хотя бы одно из условий выполняеться, результатом будет true. Оператор && отображает значение И. В этом случае если хотя бы одно из условий не выполняется, то результат будет равен false.

Для чего используется break? Почему в конструкции switch нужно использовать break для каждого case?

Break позволяет выйти из цикла в любой момент. Данная директива обязывает следующие switch конструкции выполнять проверку условия, иначе действия будут выполняться без проверки.

Данные операторы применяют для сокращения длины кода. Они выполняют арифметическое действие вместе с переназначение переменной. Существую такие операторы скоращенной арифметики:

Чем отличается Function Declaration от Function Expression?

Function Declaration - объявляется в основном потоке кода. Их можно вызывать до непосредственного объявления, так как создаются интерпретатором до выполнения кода.

Function Expression - объявляется через какое либо выражение.
Пример:
var sum = function(a, b) {
return a + b;
}

Что такое ООП? Какие ты знаешь принципы ООП?

ООП или объектно ориенторованое програмирование, при котором функции и данные группируються в "объекте". При ООП в каждом объекте описываються взаимодействие объектов между собой и со страницей браузера, а так же смена их свойств.

Что такое прототипное наследование?

Каждый объект содержит в себе ссылку на другой объект, который в свою очередь называют прототипом. Такие прототипы могут иметь свой прототип. Данная цепочка завершиться в случае, если свойство последующего прототипа будет равно null. С помощью наследования прототип может получать доступ к методам основного объекта.

Как можно привязывать обработчики к событию? Какие есть недостатки у разных способов привязки к событиям?

Существует три способа привязки обработчика к событию. Первый способ, это назначение события в HTML через атрибут onclick="...". Данный способ так же ограничен в количестве обработчиков на одно событие. При назначении больше одного, выполнятся будет только последний. Третий способ, это метод addEventListener, у него отсутсвуют недостатки предыдущих, но удаление обработчика происходит с помощью другого метода removeEventListener.

Что такое "делегирование"? В чем преимущество использования делегирования?

При наличии нескольких элементов с похожим обработчиком события, присвоение идёт их одному общему предку - в этом и заключаеться приём делегирования. Такой способ позволяет значительно сократить длину кода.

Как удалить обработчик события?

Способ удаления зависит от способа назначения события. При использовании элементов DOM, удаление происходит стандартным способом delete. Способ с методом addEventListener, будет удалён с помощью другого метода removeEventListener. Важно указать именно ту функцию-обработчик которая была назначена.

Чему равен this в функции-обработчике события? Какая есть особенность у стрелочных функций и this?

This будет ссылаться на элемент на котором он сработал. Стрелочные функции не создают собственный контекст this, а используют значение this окружающего контекста.

Что такое "захват" и "всплытие". Какую роль они играют?

При задействовании события, информация о нём проходит по dom-дереву к элементу target - захват. Обратное действие называют всплытие. Они помогают разработчику перехватить и обработать действия пользователя.

Что такое объект Event? Какую информацию он хранит?

Это объект события, в который браузер записывает детали произошедшего. Передаётся первый аргумент в обработчике.

Как отменить событие по умолчанию?

Для отмены события по умолчанию существует метод event.preventDefault().

Как остановить "всплытие" на каком-то элементе? Почему это нужно делать с осторожностью?

Для остановки всплытия используют метод event.stopPropagation(). Проблема в том, что stopPropagation убивает всякую возможность отследить событие сверху, а это бывает нужно для реализации чего-нибудь «эдакого», что к меню отношения совсем не имеет.
