Зачем нужны функции?

Функция задаёт действие, потом с помощью её вызова мы можем выполнять данное действие несколько раз и в любой части программы.

Сколько параметров можно передать функции?

Функцию можно вызвать с любым количеством аргументов.

Что случится, если передать в функцию меньше параметров, чем нужно? Объясни на примере функции add(a, b).

В данном примере, при применении функции к менее, чем 2 переменным одновременно, значение не назначенной переменной будет неопределенно ("undefined").

Возможно ли для параметра указать значение по-умолчанию? Как?

Да возможно. Нужно указать нужное значение по умолчанию для параметра в заголовке функции. Либо ввести цикл if в условие фукнции.
Пример: add(a, b=3) {
if (b === undifine) {
b=3;
} alert (a+b);

Что такое область видимости?

Область видимости определяет область в которой будет виден объект. К примеру переменные заданные в теле функции будут действовать только приделах этой функции, для других мест программы они останутся невидимыми.

Чем отличается Function Declaration от Function Expression?

Function Declaration - объявляется в основном потоке кода. Их можно вызывать до непосредственного объявления, так как создются интерпретатором до выполнения кода.
Пример:
function sum(a, b) {
return a + b;
}
Function Expression - объявляется через какое либо выражение.
Пример:
var sum = function(a, b) {
return a + b;
}

Что такое замыкание?

Это инструмент который использует возможности зоны видимости. С помощью этого приёма в функции можно вызывать переменную из области в которой находиться она сама.
Пример:
function numberGenerator() {
var num = 1;
function checkNumber() {
console.log(num);
}
}
