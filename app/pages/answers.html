Зачем нужны функции?

Функция задаёт действие, потом с помощью её вызова мы можем выполнять данное действие несколько раз и в любой части программы.

Что такое область видимости?

Область видимости определяет область в которой будет виден объект. К примеру переменные заданные в теле функции будут действовать только приделах этой функции, для других мест программы они останутся невидимыми.

Чем отличается Function Declaration от Function Expression?

Function Declaration - объявляется в основном потоке кода. Их можно вызывать до непосредственного объявления, так как создются интерпретатором до выполнения кода.
Пример:
function sum(a, b) {
return a + b;
}
Function Expression - объявляется через какое либо выражение.
Пример:
var sum = function(a, b) {
return a + b;
}

Что такое замыкание?

Это инструмент который использует возможности зоны видимости. С помощью этого приёма в функции можно вызывать переменную из области в которой находиться она сама.
Пример:
function numberGenerator() {
var num = 1;
function checkNumber() {
console.log(num);
}
}

Как можно добавить свойства или методы у объекта? Как можно удалить свойство или метод?

Для добавления свойства или метода используют запись вида объект.свойство/метод. Для удаления свойства или объекта используют оператор delete: delete объект.свойство/метод.

Как создать свойство только для чтения?

Для создания такого свойства следует использовать метод Object.defineProperty. С помощью дескрипторов данного метода мы можем задать поведение свойству. В нашем случае мы используем дескриптор writable. Он определяет возможности изменения значения свойства. Что бы свойство было только для чтения, в дескрипторе указываем false.

Что такое ООП? Какие ты знаешь принципы ООП?

ООП или объектно ориенторованое програмирование, при котором функции и данные группируються в "объекте". При ООП в каждом объекте описываються взаимодействие объектов между собой и со страницей браузера, а так же смена их свойств.

Что такое "функциональное программирование"? Чем отличается ООП и ФП? Какие есть функциональные библиотеки в JavaScript?

ФП это программирование в основе которого лежит формальная система Лямбда-исчисления. Отличие состоит в том, что в ФП вычисление производят с помощью результатов функций от исходных данных и результатов других функций. Underscore.js

Что такое прототипное наследование?

Каждый объект содержит в себе ссылку на другой объект, который в свою очередь называют прототипом. Такие прототипы могут иметь свой прототип. Данная цепочка завершиться в случае, если свойство последующего прототипа будет равно null. С помощью наследования прототип может получать доступ к методам основного объекта.

Как можно обратиться к свойству с именем my-SoMe-PROPERTY+name?

Данное имя вызовет ошибку из-за использования знаков "-" и "+". Для воизбежания такой ситуации следует использовать CamelCase. Верное имя будет выглядить вот так mySoMePROPERTYname. А образение будет иметь вид Object.mySoMePROPERTYname.

Объясни, что такое деструктуризация.

Деструктуризация - синтаксис при котором массив или объект разбивают на части и таким образом присваивают несколько переменных.
